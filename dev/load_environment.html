Load Existing Environment

Dojo includes a number of existing environments:

  • Dojo.Ant
  • Dojo.Atlas
  • Dojo.Cartpole
  • Dojo.HalfCheetah
  • Dojo.Hopper
  • Dojo.Pendulum
  • Dojo.Quadruped
  • Dojo.RaibertHopper
  • Dojo.RexHopper
  • Dojo.Walker

Specific environments can be instantiated, for example Dojo.Atlas:

env = get_environment(:atlas)
